# Release Archives

Эта ветка содержит автоматически созданные архивы проекта.

## Структура

- `latest.zip` - последняя версия архива
- `ocstore-[branch]-[timestamp]-[commit].zip` - архивы с историей
- `MANIFEST.json` - метаданные последнего релиза
- `*.sha256` - контрольные суммы для верификации

## Скачивание

Прямая ссылка на последний архив:
```
https://github.com/YOUR_USERNAME/YOUR_REPO/raw/release/latest.zip
```

Замените YOUR_USERNAME и YOUR_REPO на ваши данные.
